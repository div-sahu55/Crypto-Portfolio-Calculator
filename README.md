# Crypto-Portfolio-Calculator
A portfolio calculator for cryptocurrency and stocks investments written in C++
<p></p>
<h2>To Build</h2>

```
git clone https://github.com/div-sahu55/Crypto-Portfolio-Calculator.git
cd Crypto-Port-Calc
cmake -B build/
cd build
make
```
<h2>To Run</h2>

```
./port-calc
```
